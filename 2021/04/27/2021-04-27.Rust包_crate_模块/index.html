1 packages crate

1.1 packages

  • cargo new 创建的为包 src/main.rs 为包的同名 crate 的根
  • Cargo.toml 文件描述 crate 的构造

1.2 crate

  • 为 src/lib.rs 库文件或存在于 src/bin/ 目录下的可执行文件

2 module与作用域

  • 绝对路径要更好用一些
  • 子模块可以使用父模块中的条目
  • 父模块无法使用子模块中的私有条目
  • pub 关键字将条目标记为公共的，可以使子模块中的条目在父级模块中使用

3 关键字

3.1 super

super 上级目录

3.2 pub

pub 标记为公共的

  1. pub 放在 struct 前
  • struct 是公共的
  • struct 的字段默认是私有的(想把哪个字段设为共有在哪个字段前加pub)
  1. pub 放在 enum 前
  • enum 是公共的
  • 默认所有的变体也是公共的

4 use

  • 使用嵌套路径可以消除大量的 use 行
  • 通配符 * 可以将一个路径中所有的公共条目都引入到作用域(谨慎使用, 一般不使用)
  • 引用时最低可以到末级条目，但建议只引用到末级模块(类似于文件系统中的软链接)

4.3 pub use 重新导出

  • 使用 use 将路径(名称)导入到作用域内后，该名称在此作用域内是私有的
  • pub 可以 将 use 标记为公共的
  • 将条目引入作用域
  • 该条目可以被外部代码引入到他们的作用域

5 使用外部包

  1. 在 catgo.toml 添加依赖的包和版本
  2. use 将特定条目引入作用域
